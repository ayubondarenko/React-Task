# React-Task
This is a small client-server application where:

  Front-End: React, Redux, Redux-Saga, React-Router, Bootstrap, Webpack, Babel

  Back-End: JSON Server
  
In terminal go to [project folder]
    1: npm instal
    2: npm run build
    3: npm start
